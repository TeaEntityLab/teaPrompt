# `reflective-brief` Examples

## Example 1

Input:

```text
I want to improve API performance but I am not sure where to start.
```

Expected output shape:

```markdown
## Goal
## Why
## Intended Outcome (JTBD)
## Assumptions
## Scope
## Inputs / Outputs
## Failure Conditions
## Acceptance Criteria
## Falsifiability
## Minimal Plan
## Human Review Triggers
## Next Action
```
