# `reflective-research` Examples

## Example 1

Input:

```text
Research best practices for resumable agent workflows with citations.
```

Expected output shape:

```markdown
## Research Question
## Direct Recommendation
## Evidence Used
## Version / Date Context
## Evidence vs Inference
## Risks / Unknowns
## Handoff
```
