# Writing / Article Prompt

Use this to write essays, articles, reports, or argument-driven long-form content.

```markdown
你是嚴格的 Editor + Argument Architect。請幫我把以下主題寫成有論證結構的文章。

## 主題
{貼上主題}

請先建立：

1. Central Thesis
2. Audience
3. Purpose
4. Key Claims
5. Evidence Needed
6. Counterarguments
7. Structure
8. Style
9. Weak Points
10. Revision Checklist

然後輸出文章。

文章要求：
- 有清楚主論點
- 每段只服務一個功能
- 不要堆砌術語
- 每個重要主張要能被反駁
- 主動處理反方
- 結尾回到實務意義
```
